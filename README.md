# java-racingcar-practice1

### 기능 목록
- [ ] 자동차 대수를 입력받는다
- [ ] 시도할 횟수를 입력 받는다
- [ ] 자동차는 0~9 사이 랜덤 값 중에서 4 이상인 경우 전진할 수 있다
  - [ ] 자동차
    - [ ] 위치 정보를 가진다
    - [ ] 자동차의 전진 조건에 따라 위치 정보를 증가한다
  - [ ] 자동차 전진 조건
    - [ ] 4 이상인 경우 전진할 수 있다
  - [ ] 주사위
    - [ ] 0~9 랜덤으로 나오는 주사위
- [ ] 자동차 전진 결과를 출력한다
